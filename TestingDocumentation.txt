Interesting Bugs and Problems
During testing, a few issues were discovered and resolved:

Dropdown Menu on Mobile: Initially, the dropdown menu was not expanding properly on some mobile devices. This was fixed by adjusting the CSS for better compatibility.
Image Loading Times: Some images were loading slowly on the mobile network. This was addressed by optimizing image sizes and using responsive image techniques.
